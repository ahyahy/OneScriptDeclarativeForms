Перечисления

Перечисления (Enumeration) Описание
БазоваяЛинияТекста (TextBaseline) Указывает базовую линию при рисовании текста.
ВариантШрифта (FontVariant) Указывает значение, должен ли шрифт отображаться маленькими заглавными буквами.
ВертикальноеВыравнивание (VerticalAlign) Указывает вертикальное выравнивание содержимого элемента.
Видимость (Visibility) Указывает значение, должен ли элемент быть видимым.
ВписываниеОбъекта (ObjectFit) Указывает как содержимое элемента должно помещаться в поле, заданное по высоте и ширине.
ВыделениеПользователем (UserSelect) Указывает значение, можно ли выделить текст элемента.
ВыравниваниеОтдельных (AlignSelf) Указывает выравнивание одного или нескольких выбранных элементов внутри гибкого контейнера.
ВыравниваниеСодержимого (AlignContent) Указывает выравнивание между строками внутри гибкого контейнера.
ВыравниваниеТекста (TextAlign) Указывает выравнивание текстового содержимого.
ВыравниваниеЭлементов (AlignItems) Указывает выравнивание элементов внутри гибкого контейнера.
ГоризонтальноеВыравнивание (CssFloat) Указывает горизонтальное выравнивание элемента.
ГоризонтальноеВыравниваниеТекста (HorizontalTextAlign) Указывает горизонтальное выравнивание текста.
ГраницаСвернута (BorderCollapse) Указывает значение, должна ли граница таблицы быть свернута в одну границу.
ЖирностьШрифта (FontWeight) Указывает жирность шрифта.
ЗаливкаАнимации (AnimationFillMode) Указывает значения стилей ключевых кадров которые применяются до или после анимации.
ЗаполнениеКолонок (ColumnFill) Указывает способ заполнения колонок.
ИзменяемыйРазмер (Resize) Указывает значение, может ли пользователь изменить размер элемента.
Калибровка (BoxSizing) Указывает вписывание элемента в область.
КнопкиМыши (MouseButtons) Указывает какая кнопка мыши была нажата.
Композиция (GlobalCompositeOperation) Указывает способ рисования нового изображения на существующем изображении.
Курсор (Cursor) Указывает тип курсора, отображаемого указателем мыши.
МозаикаКартинки (BackgroundRepeat) Указывает способ повторения (мозаики) фонового изображения.
МозаикаКартинкиГраницы (BorderImageRepeat) Указывает должна ли картинка границы повторяться, закругляться или растягиваться.
Назначение (Target) Указывает где открыть связанный по ссылке документ.
Направление (Dir) Указывает направление расположения элементов внутри элемента.
НаправлениеАнимации (AnimationDirection) Указывает значение, должна ли анимация воспроизводиться в прямом или обратном порядке.
НаправлениеЭлементов (FlexDirection) Указывает направление элементов внутри флекс-контейнера.
Несвободно (Clear) Указывает положение элемента относительно плавающих объектов.
ОбластьКартинки (BackgroundOrigin) Указывает какую область фоновая картинка будет занимать внутри элемента.
ОбластьРисования (BackgroundClip) Указывает область рисования фона.
Отображать (Display) Указывает тип отображения элемента.
ОформлениеТекстаЛиния (TextDecorationLine) Указывает тип линии в текстовом оформлении.
ОформлениеТекстаСтиль (TextDecorationStyle) Указывает стиль линии в текстовом оформлении.
ПереносГибких (FlexWrap) Указывает значение, должны ли переноситься гибкие элементы.
ПереносСлов (WordWrap) Указывает как разбивать длинные неразрывные слова и переносить их на следующую строку.
ПереносТекста (TextWrap) Указывает перенос в области текста.
Переполнение (Overflow) Указывает поведение элемента при выходе содержимого за поля элемента.
ПереполнениеТекста (TextOverflow) Указывает то, что должно произойти, когда текст переполняет содержащий его элемент.
ПовторШаблона (PatternRepeat) Указывает варианты повтора шаблона.
Позиция (Position) Указывает тип позиционирования для элемента.
ПозицияСтиляСписка (ListStylePosition) Указывает позицию маркера элемента списка.
ПозицияФормы (FormPosition) Позиция формы при загрузке.
ПоложениеЗаголовка (CaptionSide) Указывает положение заголовка таблицы.
ПоложениеКартинки (BackgroundPosition) Указывает начальную позицию фонового изображения.
Пробелы (WhiteSpace) Возвращает или задает способ обработки табуляции, разрывов строк и пробелов в тексте.
ПрописныеТекста (TextTransform) Указывает прописные (заглавные) буквы текста.
ПустыеЯчейки (EmptyCells) Указывает показывать ли границу и фон пустых ячеек.
Разлиновка (Rules) Указывает как отображать границы между ячейками.
РазмерКартинки (BackgroundSize) Указывает размер фонового изображения.
РазмещениеВТаблице (TableLayout) Указывает способ расположения ячеек, строк и колонок таблицы.
РасположениеСодержимого (JustifyContent) Указывает выравнивание между элементами внутри гибкого контейнера, когда элементы не используют все доступное пространство.
СвойствоПерехода (TransitionProperty) Указывает свойство для которого предназначен эффект перехода.
Состояние (AnimationPlayState) Указывает значение, запущена или приостановлена ​​анимация.
СтильГраницы (BorderStyle) Указывает стиль границы.
СтильКонтура (OutlineStyle) Указывает стиль контура вокруг элемента.
СтильКонцовЛинии (LineCap) Указывает стиль концов линии.
СтильРазделителяКолонок (ColumnRuleStyle) Указывает стиль разделителя колонок.
СтильСдвига (TransformStyle) Указывает способ отображения вложенных элементов в трехмерном пространстве.
СтильУглаПересечения (LineJoin) Указывает тип угла, создаваемого при пересечении двух линий.
СтильШрифта (FontStyle) Указывает тип шрифта.
ТипВыбранногоФайла (SelectedFileType) Указывает тип файла при выборе.
ТипМаркера (MarkerType) Указывает тип маркера для упорядоченного списка.
ТипМеню (MenuType) Указывает тип меню при создании.
ТипОтображения (DisplayType) Указывает тип отображения элемента.
ТипСтиляСписка (ListStyleType) Указывает тип маркера элемента списка.
ТипЭлементаМеню (MenuItemType) Указывает тип элемента меню при создании.
ФоновоеВложение (BackgroundAttachment) Указывает значение, является ли фоновое изображение фиксированным или прокручивается вместе со страницей.
ФункцияПерехода (TransitionTimingFunction) Указывает кривую скорости эффекта перехода.
ФункцияСинхронизации (AnimationTimingFunction) Указывает кривую скорости анимации.
Цвет (Color) Представляет цвет.
ШиринаГраницы (BorderWidth) Указывает ширину границы.
