Библиотека DeclarativeForms События

События

События (Events) Описание
Бросить (Drop) (Элемент) Возвращает или задает код для события, которое генерируется при отпускании перетаскиваемого элемента на целевой области.
Ввод (Input) (Элемент) Возвращает или задает код, когда элемент получает ввод данных от пользователя.
ДвойноеНажатие (DoubleClick) (Элемент) Возвращает или задает код, когда элемент дважды щелкнут.
Загружена (Loaded) (Форма) Возвращает или задает код, когда форма полностью загружена.
КонецПеретаскивания (DragEnd) (Элемент) Возвращает или задает код, когда перетаскивание элемента завершено.
МышьНадЭлементом (MouseOver) (Элемент) Возвращает или задает код для выполнения когда указатель мыши наведен на элемент.
МышьПокинулаЭлемент (MouseOut) (Элемент) Возвращает или задает код для выполнения, когда указатель мыши покидает пределы элемента.
НадЦелью (DragEnter) (Элемент) Возвращает или задает код, когда элемент входит в границы целевой области.
Нажатие (Click) (Лоток) Возвращает или задает код для выполнения при нажатии элемента управления.
Нажатие (Click) (Элемент) Возвращает или задает код для выполнения при нажатии элемента управления.
Нажатие (Click) (ЭлементМеню) Возвращает или задает код для выполнения при нажатии элемента меню.
НачалоПеретаскивания (DragStart) (Элемент) Возвращает или задает код, когда начинается перетаскивание элемента.
ПриИзменении (Change) (ВыборФайла) Возвращает или задает код, когда пользователь произвел выбор файлов.
ПриИзменении (Change) (ПолеСписка) Возвращает или задает код, когда пользователь произвел выбор в поле списка.
ПриИзменении (Change) (Флажок) Возвращает или задает код при изменении состояние флажка.
ПриИзменении (Change) (Элемент) Возвращает или задает код при изменении значения элемента формы.
ПриНажатииМыши (MouseDown) (Элемент) Возвращает или задает код для выполнения при нажатии кнопки мыши, когда указатель мыши находится на элементе.
ПриОтпусканииМыши (MouseUp) (Элемент) Возвращает или задает код для выполнения при отпускании кнопки мыши, когда указатель мыши находится на элементе.
ПриПеремещенииМыши (MouseMove) (Элемент) Возвращает или задает код для выполнения, когда указатель мыши переместился.
ПриПрокручивании (Scroll) (Форма) Возвращает или задает код для выполнения, при перемещении ползунка полосы прокрутки, с помощью мыши или клавиатурой.
ПриСрабатыванииТаймера (Tick) (Таймер) Возвращает или задает код для выполнения по истечении указанного интервала таймера.
РазмерИзменен (Resize) (Форма) Возвращает или задает код, когда размера окна изменен.
ФокусПолучен (Focused) (Элемент) Возвращает или задает код для выполнения, когда элемент получает фокус.
ФокусПотерян (LostFocus) (Элемент) Возвращает или задает код для выполнения, когда элемент теряет фокус.
ЦельПокинута (DragLeave) (Элемент) Возвращает или задает код для события, которое генерируется, когда элемент покидает целевую область.
