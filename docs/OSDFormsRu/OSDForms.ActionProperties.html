Действие (Action) Свойства

Свойства

ИмяМетода (MethodName) Возвращает или задает имя метода обработчика события для контрола.
Параметр (Parameter) Возвращает или задает объект, заданный при установке обработчика события, для передачи в обработчик события.
Сценарий (Script) Возвращает или задает сценарий как объект с типом значения ScriptEngine.Machine.IRuntimeContextInstance в котором находится обработчик события.
