ДополнительныйБлок (AdditionalDiv) События

События

ДвойноеНажатие (DoubleClick) Возвращает или задает код, когда элемент дважды щелкнут.
МышьНадЭлементом (MouseOver) Возвращает или задает код для выполнения когда указатель мыши наведен на элемент.
МышьПокинулаЭлемент (MouseOut) Возвращает или задает код для выполнения, когда указатель мыши покидает пределы элемента.
Нажатие (Click) Возвращает или задает код для выполнения при нажатии элемента управления.
ПриОтпусканииМыши (MouseUp) Возвращает или задает код для выполнения при отпускании кнопки мыши, когда указатель мыши находится на элементе.
