ВыравниваниеСодержимого (AlignContent) Перечисление

Описание

Указывает выравнивание между строками внутри гибкого контейнера.

Члены

Имя члена Описание Значение
ГибкийВконце (FlexEnd) Расположение в конце контейнера. flex-end
ГибкийВначале (FlexStart) Расположение в начале контейнера. flex-start
ПробелВокруг (SpaceAround) Расположение с пробелом до, между и после строк. space-around
ПробелМежду (SpaceBetween) Расположение с пробелом между строками. space-between
Растянуто (Stretch) Элементы растягиваются по размеру контейнера. Это значение по умолчанию. stretch
Центр (Center) Расположение в центре контейнера. center
