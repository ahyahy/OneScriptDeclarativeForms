ПараметрыАнимации (AnimationOptions) Свойства

Свойства

Длительность (Duration) Возвращает количество миллисекунд длительности одного цикла анимации.
Задержка (Delay) Возвращает задержку перед началом анимации.
ЗадержкаПосле (EndDelay) Возвращает задержку после завершения анимации в миллисекундах.
Заливка (Fill) Возвращает значения стилей ключевых кадров которые применяются до или после анимации.
КоличествоПовторов (Iterations) Возвращает количество повторений анимации.
Направление (Direction) Возвращает значение, должна ли анимация воспроизводиться в прямом или обратном порядке.
ФункцияСинхронизации (Easing) Возвращает функцию синхронизации для анимации.
