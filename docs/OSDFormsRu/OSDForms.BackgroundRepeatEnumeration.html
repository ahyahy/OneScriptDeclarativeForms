МозаикаКартинки (BackgroundRepeat) Перечисление

Описание

Указывает способ повторения (мозаики) фонового изображения.

Члены

Имя члена Описание Значение
НеПовторять (NoRepeat) Фоновое изображение не повторяется. no-repeat
Повторять (Repeat) Фоновое изображение повторяется и по вертикали, и по горизонтали. Это значение по умолчанию. repeat
ПовторятьВертикально (RepeatY) Фоновое изображение повторяется только вертикально. repeat-y
ПовторятьГоризонтально (RepeatX) Фоновое изображение повторяется только по горизонтали. repeat-x
