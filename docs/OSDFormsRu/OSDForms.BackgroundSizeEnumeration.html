РазмерКартинки (BackgroundSize) Перечисление

Описание

Указывает размер фонового изображения.

Члены

Имя члена Описание Значение
Авто (Auto) Фоновое изображение сохраняет ширину и высоту. Это значение по умолчанию. auto
Вместить (Contain) Изображение масштабируется до максимального размера так, чтобы его ширина и высота могли поместиться внутри области содержимого. contain
Накрыть (Cover) Фоновое изображение масштабируется как можно больше, чтобы область фона была полностью покрыта фоновым изображением. Некоторые части фонового изображения могут быть не видны в области позиционирования фона. cover
