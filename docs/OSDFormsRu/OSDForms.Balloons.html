Уведомления (Balloons) Класс

Описание

Представляет шаблон для окна уведомления с указанным заголовком и текстом.

Этот класс в своих свойствах задает содержимое и вид окна всплывающего уведомления. Само окно уведомления представляет объект другого класса и показывается пользователю методом Уведомления.Показать (Balloons.Show).

Можно создав один шаблон показывать его на экране много раз. Изменяя свойства шаблона перед показом можно изменять вид уведомлений.

Настроив вид шаблона с помощью цвета, размера окна и измененного шрифта можно создавать тематические уведомления. Например, уведомления с красным фоном - важные, с серым - второстепенные.
