Уведомления (Balloons) Конструктор

Синтаксис

Уведомления()

Параметры

Описание

Создаёт новый экземпляр класса Уведомления (Balloons).

Примечание

Важно: При установке свойства ДекларативныеФормы.ОткрытьВБраузере (DeclarativeForms.OpenInBrowser) в значение Истина этот элемент управления не работает.

Важно: В ОС Linux включите группировку кнопок в панели задач, чтобы избежать появления там множества кнопок от каждого нового уведомления.

Пример

Уведомления1 = ДФ.Уведомления();
