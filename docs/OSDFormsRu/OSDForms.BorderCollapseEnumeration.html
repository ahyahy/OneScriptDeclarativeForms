ГраницаСвернута (BorderCollapse) Перечисление

Описание

Указывает значение, должна ли граница таблицы быть свернута в одну границу.

Члены

Имя члена Описание Значение
Раздельно (Separate) Рисуются отдельные границы. Это значение по умолчанию. separate
Свернуто (Collapse) Границы между элементами ячеек таблицы не рисуются. collapse
