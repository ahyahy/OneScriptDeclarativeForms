СтильГраницы (BorderStyle) Перечисление

Описание

Указывает стиль границы.

Члены

Имя члена Описание Значение
Граница (Inset) Граница 3D-вставки. Эффект зависит от значения цвета границы. inset
Двойная (Double) Двойная сплошная граница. Ширина такая же, как значение ширины границы. double
НачальнаяГраница (Outset) Начальная граница 3D. Эффект зависит от значения цвета границы. outset
Отсутствие (None) Не определяет границ. Это значение по умолчанию. none
Ребристая (Ridge) 3D ребристая граница. Эффект зависит от значения цвета границы. ridge
Скрыто (Hidden) То же, что и none, за исключением разрешения конфликтов границ для элементов таблицы. hidden
Сплошная (Solid) Сплошная граница. solid
ТочкаПробел (Dotted) Точки с пробелами между ними. dotted
Трехмерная (Groove) Трехмерная рифленая граница. Эффект зависит от значения цвета границы. groove
ЧертаПробел (Dashed) Короткие чёрточки с пробелами между ними. dashed
