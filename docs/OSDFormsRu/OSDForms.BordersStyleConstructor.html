СтильГраниц (BordersStyle) Конструктор

Синтаксис

СтильГраниц(<Верхняя>, <Правая>, <Нижняя>, <Левая>)

Параметры

Верхняя (необязательный)
Тип: СтильГраницы (BorderStyle).
Стиль верхней границы.
Правая (необязательный)
Тип: СтильГраницы (BorderStyle).
Стиль правой границы.
Нижняя (необязательный)
Тип: СтильГраницы (BorderStyle).
Стиль нижней границы.
Левая (необязательный)
Тип: СтильГраницы (BorderStyle).
Стиль левой границы.

Описание

Создаёт новый экземпляр класса СтильГраниц (BordersStyle).

Примечание

Пример

СтильГраниц1 = ДФ.СтильГраниц();
