ШиринаГраниц (BordersWidth) Конструктор

Синтаксис

ШиринаГраниц(<Верхняя>, <Правая>, <Нижняя>, <Левая>)

Параметры

Верхняя (необязательный)
Тип: ШиринаГраницы (BorderWidth); Число.
Ширина верхней границы.
Правая (необязательный)
Тип: ШиринаГраницы (BorderWidth); Число.
Ширина правой границы.
Нижняя (необязательный)
Тип: ШиринаГраницы (BorderWidth); Число.
Ширина нижней границы.
Левая (необязательный)
Тип: ШиринаГраницы (BorderWidth); Число.
Ширина левой границы.

Описание

Создаёт новый экземпляр класса ШиринаГраниц (BordersWidth).

Примечание

Пример

ШиринаГраниц1 = ДФ.ШиринаГраниц();
