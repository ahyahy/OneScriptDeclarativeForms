Тень.Цвет (BoxShadow.Color) Свойство

Использование

Чтение и запись.

Значение

Тип: Цвет (Color).

Цвет тени. Значение по умолчанию - черный.

Примечание

Пример

Тень1.Цвет = ДФ.Цвет.Серебристый;
