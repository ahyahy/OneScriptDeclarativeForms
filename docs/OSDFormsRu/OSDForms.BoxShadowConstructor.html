Тень (BoxShadow) Конструктор

Синтаксис

Тень(<Икс>, <Игрек>, <Размытие>, <Величина>, <Цвет>, <Внутренняя>)

Параметры

Икс (необязательный)
Тип: Число.
Горизонтальное положение тени. Допускаются отрицательные значения.
Игрек (необязательный)
Тип: Число.
Вертикальное положение тени. Допускаются отрицательные значения.
Размытие (необязательный)
Тип: Число.
Расстояние размытия.
Величина (необязательный)
Тип: Число.
Размер тени.
Цвет (необязательный)

Тип: Цвет (Color).

Цвет тени. Значение по умолчанию - черный.
Внутренняя (необязательный)

Тип: Булево.

Изменяет тень с внешней на внутреннюю.

Описание

Создаёт новый экземпляр класса Тень (BoxShadow).

Примечание

Пример

Блок1.Стиль.Тень = ДФ.Тень(30, 30, 10, 10, ДФ.Цвет.Серебристый);
