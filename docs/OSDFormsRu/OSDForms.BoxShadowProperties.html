Тень (BoxShadow) Свойства

Свойства

Величина (Spread) Размер тени.
Внутренняя (Inset) Изменяет тень с внешней на внутреннюю.
Игрек (Y) Вертикальное положение тени. Допускаются отрицательные значения.
Икс (X) Горизонтальное положение тени. Допускаются отрицательные значения.
Размытие (Blur) Расстояние размытия.
Цвет (Color) Цвет тени. Значение по умолчанию - черный.
