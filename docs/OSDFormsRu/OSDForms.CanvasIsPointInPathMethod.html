Холст.ТочкаПути (Canvas.IsPointInPath) Метод

Синтаксис

ТочкаПути(<Икс>, <Игрек>)

Параметры

Икс (обязательный)
Тип: Число.
x координата точки.
Игрек (обязательный)
Тип: Число.
y координата точки.

Возвращаемое значение

Тип: Булево.

Описание

Возвращает Истина, если указанная точка находится на текущем рисунке.
