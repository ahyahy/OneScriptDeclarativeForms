Ячейка.ДиапазонКолонок (Cell.ColSpan) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает количество колонок, которые должна занимать ячейка таблицы.

Примечание

Пример

Ячейка1.ДиапазонКолонок = 2;
Полный пример кода 

Копировать     Выделить всё


Перем ДФ;
Перем Ячейка1;

Процедура Свойство() Экспорт
	Сообщить("Ячейка1.ДиапазонКолонок = " + Ячейка1.ДиапазонКолонок);
КонецПроцедуры

Процедура Форма1_Загружена() Экспорт
	ДФ.ПолучитьСвойство(Ячейка1, "ДиапазонКолонок", ДФ.Действие(ЭтотОбъект, "Свойство"));
