Ячейка.ДиапазонСтрок (Cell.RowSpan) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает количество строк, которые должна занимать ячейка таблицы.

Примечание

Пример

Ячейка1.ДиапазонСтрок = 2;
