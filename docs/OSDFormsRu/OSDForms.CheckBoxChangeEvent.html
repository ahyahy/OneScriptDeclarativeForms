Флажок.ПриИзменении (CheckBox.Change) Событие

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код при изменении состояние флажка.

Примечание

Аргументы события

Значение (Value) Возвращает значение свойства элемента.

Пример

Флажок1.ПриИзменении = ДФ.Действие(ЭтотОбъект, "Флажок1_ПриИзменении");
