Флажок (CheckBox) События

События

Ввод (Input) Возвращает или задает код, когда элемент получает ввод данных от пользователя.
ДвойноеНажатие (DoubleClick) Возвращает или задает код, когда элемент дважды щелкнут.
КонецПеретаскивания (DragEnd) Возвращает или задает код, когда перетаскивание элемента завершено.
МышьНадЭлементом (MouseOver) Возвращает или задает код для выполнения когда указатель мыши наведен на элемент.
МышьПокинулаЭлемент (MouseOut) Возвращает или задает код для выполнения, когда указатель мыши покидает пределы элемента.
Нажатие (Click) Возвращает или задает код для выполнения при нажатии элемента управления.
НачалоПеретаскивания (DragStart) Возвращает или задает код, когда начинается перетаскивание элемента.
ПриИзменении (Change) Возвращает или задает код при изменении состояние флажка.
ФокусПолучен (Focused) Возвращает или задает код для выполнения, когда элемент получает фокус.
ФокусПотерян (LostFocus) Возвращает или задает код для выполнения, когда элемент теряет фокус.
