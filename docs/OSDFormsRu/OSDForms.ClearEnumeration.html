Несвободно (Clear) Перечисление

Описание

Указывает положение элемента относительно плавающих объектов.

Члены

Имя члена Описание Значение
Лево (Left) С левой стороны элемента запрещены плавающие объекты. left
ЛевоПраво (Both) Плавающие объекты не допускаются с обеих сторон от элемента. both
Отсутствие (None) Допускаются плавающие объекты по обе стороны от элемента. Это значение по умолчанию. none
Право (Right) С правой стороны элемента запрещены плавающие объекты. right
