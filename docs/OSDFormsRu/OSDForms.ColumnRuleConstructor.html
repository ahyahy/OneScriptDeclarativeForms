РазделительКолонок (ColumnRule) Конструктор

Синтаксис

РазделительКолонок(<Ширина>, <Стиль>, <Цвет>)

Параметры

Ширина (необязательный)
Тип: ШиринаГраницы (BorderWidth); Число.
Возвращает или задает ширину разделителя колонок.
Стиль (необязательный)
Тип: СтильГраницы (BorderStyle).
Возвращает или задает стиль разделителя колонок.
Цвет (необязательный)
Тип: Цвет (Color).
Возвращает или задает цвет разделителя колонок.

Описание

Создаёт новый экземпляр класса РазделительКолонок (ColumnRule).

Примечание

Пример

РазделительКолонок1 = ДФ.РазделительКолонок();
РазделительКолонок2 = ДФ.РазделительКолонок(ДФ.ШиринаГраницы.Средняя, ДФ.СтильГраницы.Двойная, ДФ.Цвет.Аквамариновый);
