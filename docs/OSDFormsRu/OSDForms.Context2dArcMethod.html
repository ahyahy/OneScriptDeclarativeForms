Контекст2d.Дуга (Context2d.Arc) Метод

Синтаксис

Дуга(<ИксЦентра>, <ИгрекЦентра>, <Радиус>, <НачальныйУгол>, <КонечныйУгол>, <Направление>)

Параметры

ИксЦентра (обязательный)
Тип: Число.
x координата центра круга.
ИгрекЦентра (обязательный)
Тип: Число.
y координата центра круга.
Радиус (обязательный)
Тип: Число.
Радиус круга.
НачальныйУгол (обязательный)
Тип: Число.
Начальный угол в радианах. 0 соответствует 3-часовому положению окружности дуги.
КонечныйУгол (обязательный)
Тип: Число.
Конечный угол, в радианах.
Направление (необязательный)
Тип: Булево.
Указывает, должен ли рисунок быть расположен против часовой стрелки или по часовой стрелке. По умолчанию установлено значение Ложь указывающее по часовой стрелке.

Возвращаемое значение

Описание

Добавляет дугу или кривую (круг или части круга).

Примечание

Пример

Контекст2d1.Дуга(100, 75, 50, 0, 2 * ДФ.Математика().Пи);
