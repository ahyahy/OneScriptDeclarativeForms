Контекст2d.ДобавитьКубическуюБизье (Context2d.BezierCurveTo) Метод

Синтаксис

ДобавитьКубическуюБизье(<ПерваяИкс>, <ПерваяИгрек>, <ВтораяИкс>, <ВтораяИгрек>, <КонечнаяИкс>, <КонечнаяИгрек>)

Параметры

ПерваяИкс (обязательный)
Тип: Число.
x координата первой контрольной точки Безье.
ПерваяИгрек (обязательный)
Тип: Число.
y координата первой контрольной точки Безье.
ВтораяИкс (обязательный)
Тип: Число.
x координата второй контрольной точки Безье.
ВтораяИгрек (обязательный)
Тип: Число.
y координата второй контрольной точки Безье.
КонечнаяИкс (обязательный)
Тип: Число.
x координата конечной точки.
КонечнаяИгрек (обязательный)
Тип: Число.
y координата конечной точки.

Возвращаемое значение

Описание

Добавляет кубическую кривую Безье.

Примечание

Пример

Контекст2d1.ДобавитьКубическуюБизье(20, 100, 200, 100, 200, 20);
