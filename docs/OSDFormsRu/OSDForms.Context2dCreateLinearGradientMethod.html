Контекст2d.СоздатьЛинейныйГрадиент (Context2d.CreateLinearGradient) Метод

Синтаксис

СоздатьЛинейныйГрадиент(<ИксНачала>, <ИгрекНачала>, <ИксКонца>, <ИгрекКонца>)

Параметры

ИксНачала (обязательный)
Тип: Число.
x координата начальной точки градиента.
ИгрекНачала (обязательный)
Тип: Число.
y координата начальной точки градиента.
ИксКонца (обязательный)
Тип: Число.
x координата конечной точки градиента.
ИгрекКонца (обязательный)
Тип: Число.
y координата конечной точки градиента.

Возвращаемое значение

Описание

Создает линейный градиент для использования в содержимом холста.

Примечание

Пример

ЛинейныйГрадиент1 = Холст1.Контекст2d.СоздатьЛинейныйГрадиент(0, 0, 170, 0);

ПодключитьВнешнююКомпоненту("C:\444\111\DeclarativeForms\DeclarativeForms\bin\Debug\DeclarativeForms.dll");
ДФ = Новый ДекларативныеФормы();
