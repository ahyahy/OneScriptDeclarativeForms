Контекст2d.СоздатьРадиальныйГрадиент (Context2d.CreateRadialGradient) Метод

Синтаксис

СоздатьРадиальныйГрадиент(<ИксНачального>, <ИгрекНачального>, <РадиусНачального>, <ИксКонечного>, <ИгрекКонечного>, <РадиусКонечного>)

Параметры

ИксНачального (обязательный)
Тип: Число.
x координата начального круга градиента.
ИгрекНачального (обязательный)
Тип: Число.
y координата начального круга градиента.
РадиусНачального (обязательный)
Тип: Число.
Радиус начального круга.
ИксКонечного (обязательный)
Тип: Число.
x координата конечного круга градиента.
ИгрекКонечного (обязательный)
Тип: Число.
y координата конечного круга градиента.
РадиусКонечного (обязательный)
Тип: Число.
Радиус конечного круга.

Возвращаемое значение

Описание

Создает радиальный или круговой градиент.

Примечание

Пример

РадиальныйГрадиент1 = Холст1.Контекст2d.СоздатьРадиальныйГрадиент(75, 50, 5, 90, 60, 100);
