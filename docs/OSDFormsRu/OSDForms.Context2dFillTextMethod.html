Контекст2d.ЗаполнитьТекст (Context2d.FillText) Метод

Синтаксис

ЗаполнитьТекст(<Текст>, <ИксНачала>, <ИгрекНачала>, <МаксимальнаяШирина>)

Параметры

Текст (обязательный)
Тип: Строка.
Текст.
ИксНачала (обязательный)
Тип: Число.
x координата начала текста.
ИгрекНачала (обязательный)
Тип: Число.
y координата начала текста.
МаксимальнаяШирина (обязательный)
Тип: Число.
Максимальная ширина текста.

Возвращаемое значение

Описание

Рисует залитый текст на холсте.

Примечание

Пример

Контекст2d1.ЗаполнитьТекст("OneScript", 150, 50);
