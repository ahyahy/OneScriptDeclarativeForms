Контекст2d.Замостить (Context2d.Pave) Метод

Синтаксис

Замостить(<Рисунок>, <ПовторШаблона>, <Икс>, <Игрек>, <Ширина>, <Высота>)

Параметры

Рисунок (обязательный)
Тип: Произвольный.
Элемент, который будет использоваться.
ПовторШаблона (обязательный)
Тип: ПовторШаблона (PatternRepeat).
Одно из значений перечисления ПовторШаблона (PatternRepeat).
Икс (обязательный)
Тип: Число.
x координата левого верхнего угла начала заполнения на холсте.
Игрек (обязательный)
Тип: Число.
y координата левого верхнего угла начала заполнения на холсте.
Ширина (обязательный)
Тип: Число.
Ширина заполняемого прямоугольника на холсте.
Высота (обязательный)
Тип: Число.
Высота заполняемого прямоугольника на холсте.

Возвращаемое значение

Описание

Повторяет указанный элемент в указанном направлении.

Примечание

Пример

Контекст2d1.Замостить(Изображение1, ДФ.ПовторШаблона.Повторять, 0, 0, 550, 300);
