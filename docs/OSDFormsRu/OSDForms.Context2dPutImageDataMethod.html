Контекст2d.ПоместитьДанныеРисунка (Context2d.PutImageData) Метод

Синтаксис

ПоместитьДанныеРисунка(<ДанныеРисунка>, <ИксРисунка>, <ИгрекРисунка>, <ИксИзвлеченного>, <ИгрекИзвлеченного>, <ШиринаИзвлеченного>, <ВысотаИзвлеченного>)

Параметры

ДанныеРисунка (обязательный)
ДанныеРисунка ImageData
Объект ДанныеРисунка (ImageData).
ИксРисунка (обязательный)
Тип: Число.
x координата изображения.
ИгрекРисунка (обязательный)
Тип: Число.
y координата изображения.
ИксИзвлеченного (необязательный)
Тип: Число.
x координата. По умолчанию - 0.
ИгрекИзвлеченного (необязательный)
Тип: Число.
y координата. По умолчанию - 0.
ШиринаИзвлеченного (необязательный)
Тип: Число.
Ширина. По умолчанию - ширина извлеченного изображения.
ВысотаИзвлеченного (необязательный)
Тип: Число.
Высота. По умолчанию - высота извлеченного изображения.

Возвращаемое значение

Описание

Помещает данные изображения из указанного объекта ДанныеРисунка (ImageData) обратно на Контекст2d.

Примечание

Пример

Контекст2d1.ПоместитьДанныеРисунка(ДанныеРисунка2, 10, 10);
