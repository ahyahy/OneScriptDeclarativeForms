Контекст2d.Сохранить (Context2d.Save) Метод

Синтаксис

Сохранить()

Возвращаемое значение

Описание

Сохраняет состояние текущего контекста рисования и всех его атрибутов.

Примечание

Пример

Контекст2d1.Сохранить();
