Контекст2d.Масштабировать (Context2d.Scale) Метод

Синтаксис

Масштабировать(<Ширина>, <Высота>)

Параметры

Ширина (обязательный)
Тип: Число.
Масштабирует ширину.
Высота (обязательный)
Тип: Число.
Масштабирует высоту.

Возвращаемое значение

Описание

Масштабирует текущий рисунок увеличивая или уменьшая.

Примечание

Коэффициенты соотносятся так: 1=100%, 0,5=50%, 2=200%.

Пример

Контекст2d1.Масштабировать(1.5, 4);
