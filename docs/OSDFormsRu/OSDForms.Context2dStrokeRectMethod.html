Контекст2d.РисоватьПрямоугольник (Context2d.StrokeRect) Метод

Синтаксис

РисоватьПрямоугольник(<Икс>, <Игрек>, <Ширина>, <Высота>)

Параметры

Икс (обязательный)
Тип: Число.
x координата верхнего левого угла прямоугольника.
Игрек (обязательный)
Тип: Число.
y координата верхнего левого угла прямоугольника.
Ширина (обязательный)
Тип: Число.
Ширина прямоугольника.
Высота (обязательный)
Тип: Число.
Высота прямоугольника.

Возвращаемое значение

Описание

Рисует прямоугольник без заливки.

Примечание

Пример

Контекст2d1.РисоватьПрямоугольник(5, 5, 25, 15);
