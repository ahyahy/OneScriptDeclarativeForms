Контекст2d.РисоватьТекст (Context2d.StrokeText) Метод

Синтаксис

РисоватьТекст(<Текст>, <ИксНачала>, <ИгрекНачала>, <МаксимальнаяШирина>)

Параметры

Текст (обязательный)
Тип: Строка.
Текст.
ИксНачала (обязательный)
Тип: Число.
x координата начала текста.
ИгрекНачала (обязательный)
Тип: Число.
y координата начала текста.
МаксимальнаяШирина (необязательный)
Тип: Число.
Максимальная ширина текста.

Возвращаемое значение

Описание

Рисует текст на холсте.

Примечание

Пример

Контекст2d1.РисоватьТекст("Big smile!", 10, 90);
