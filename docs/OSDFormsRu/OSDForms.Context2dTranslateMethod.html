Контекст2d.Преобразовать (Context2d.Translate) Метод

Синтаксис

Преобразовать(<Икс>, <Игрек>)

Параметры

Икс (обязательный)
Тип: Число.
Значение, добавляемое к координате x.
Игрек (обязательный)
Тип: Число.
Значение, добавляемое к координате y.

Возвращаемое значение

Описание

Переназначает позицию рисунка.

Примечание

Пример

Контекст2d1.Преобразовать(70, 70);
