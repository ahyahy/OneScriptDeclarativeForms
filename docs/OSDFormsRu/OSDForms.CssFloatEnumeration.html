ГоризонтальноеВыравнивание (CssFloat) Перечисление

Описание

Указывает горизонтальное выравнивание элемента.

Члены

Имя члена Описание Значение
Лево (Left) Элемент будет плавать слева от родителя. left
Отсутствие (None) Элемент не является плавающим. Это значение по умолчанию. none
Право (Right) Элемент будет плавать справа в родителя. right
