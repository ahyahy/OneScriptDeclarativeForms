ДекларативныеФормы.Цвет (DeclarativeForms.Color) Метод

Синтаксис

Цвет()

Цвет(<ИмяЦвета>)

Цвет(<Красный>, <Зеленый>, <Синий>)

Параметры

ИмяЦвета
Тип: Строка.
Название цвета.
Красный
Тип: Число.
Значение красной составляющей цвета.
Зеленый
Тип: Число.
Значение зеленой составляющей цвета.
Синий
Тип: Число.
Значение синей составляющей цвета.

Возвращаемое значение

Тип: Произвольный.

Описание

Возвращает значение цвета в виде строки.

Примечание

Пример

Цвет1 = ДФ.Цвет("БледноБирюзовый");
Цвет2 = ДФ.Цвет();
Цвет3 = ДФ.Цвет(175, 238, 238);
