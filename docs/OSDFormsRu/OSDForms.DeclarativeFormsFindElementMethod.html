ДекларативныеФормы.НайтиЭлемент (DeclarativeForms.FindElement) Метод

Синтаксис

НайтиЭлемент(<ИмяЭлемента>)

Параметры

ИмяЭлемента (обязательный)
Тип: Строка.
Уникальное имя элемента управления формы, сгенерированное при его создании.

Возвращаемое значение

Тип: Произвольный.

Элемент управления формы.

Описание

Возвращает элемент управления формы, найденный по его имени.

Примечание

Пример

Сообщить("Кнопка1.Текст = " + ДФ.НайтиЭлемент(Кнопка1.КлючЭлемента).Текст);
