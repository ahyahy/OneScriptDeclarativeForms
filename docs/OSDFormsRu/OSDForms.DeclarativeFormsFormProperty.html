ДекларативныеФормы.Форма (DeclarativeForms.Form) Свойство

Использование

Только чтение.

Значение

Тип: Форма (Form).

Возвращает окно, которое составляет пользовательский интерфейс приложения.

Примечание

Свойство возвращает форму, которая создана одновременно с созданием экземпляра класса ДекларативныеФормы. Эта форма главная и при её закрытии завершится работа всего приложения. Остальные формы могут быть созданы и закрыты не вызывая закрытия приложения.

Пример

Форма1 = ДФ.Форма;
