ДекларативныеФормы.ЗагрузитьСценарии (DeclarativeForms.LoadScripts) Метод

Синтаксис

ЗагрузитьСценарии(<Каталог>)

Параметры

Каталог (обязательный)
Тип: Строка.
Каталог поиска сценариев. Если используется обозначение ".\" или "..\" или "./" или "../", поис будет проводиться в каталогах Класс и Модули.

Возвращаемое значение

Тип: Структура.

Структура, содержащая подключенные сценарии. Ключ элемента совпадает с именем файла подключенного сценария (без расширения).

Описание

Находит файлы сценария по указанному пути, вставляет их в структуру, которую подключает к текущей системе типов.

Примечание

Пример

Путь = Новый Файл(СтартовыйСценарий().Источник).Путь;
СценарииИзos = ДФ.ЗагрузитьСценарии(Путь + "os"); // И станут доступны сценарии из каталога (Путь + "os").
