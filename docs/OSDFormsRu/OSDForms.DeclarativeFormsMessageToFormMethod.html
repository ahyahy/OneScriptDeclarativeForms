ДекларативныеФормы.СообщениеФорме (DeclarativeForms.MessageToForm) Метод

Синтаксис

СообщениеФорме(<СтрокаФункций>)

Параметры

СтрокаФункций (обязательный)
Тип: Строка.
Строка с перечисленными через точку с запятой функциями для выполнения в форме посредством кода js.

Возвращаемое значение

Описание

Посылает сообщение форме через постоянно подключенного клиента.

Примечание

Пример

ДФ.СообщениеФорме("mapKeyEl.get('" + Кнопка1.КлючЭлемента + "').style['backgroundColor'] = 'rgb(175, 238, 238)'");
