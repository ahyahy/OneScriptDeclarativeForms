ДекларативныеФормы (DeclarativeForms) Методы

Методы

ВвестиГлобальноеСвойство (InjectGlobalProperty) Вводит глобальное свойство в область видимости загружаемых скриптов.
Выполнить (Execute) Вызывает заданный метод.
ЗавершитьРаботу (Exit) Завершает работу приложения.
ЗагрузитьСценарии (LoadScripts) Находит файлы сценария по указанному пути, вставляет их в структуру, которую подключает к текущей системе типов.
НайтиЭлемент (FindElement) Возвращает элемент управления формы, найденный по его имени.
ПолучитьСвойство (GetObjectProperty) Получает для указанного элемента указанное свойство, которое будет доступно для обработки в обработчике события переданном в параметре Обработчик.
СообщениеФорме (MessageToForm) Посылает сообщение форме через постоянно подключенного клиента.
Цвет (Color) Создаёт новый экземпляр класса Цвет (Color) или возвращает значение цвета в виде строки.
