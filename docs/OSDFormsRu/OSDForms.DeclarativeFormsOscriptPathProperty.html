ДекларативныеФормы.oscriptПуть (DeclarativeForms.OscriptPath) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает путь до файла oscript.exe.

Примечание

По умолчанию программа будет искать файл oscript.exe в подкаталоге \oscript\bin каталога стартового сценария. Вы можете задать значение свойства по своему усмотрению.

Можете скопировать в \oscript\ всё, что внутри каталога с установленным OneScript, но минимальный набор файлов представлен на рисунке ниже.

Пример

ДФ.oscriptПуть = "C:\Program Files\OneScript\bin\oscript.exe";
