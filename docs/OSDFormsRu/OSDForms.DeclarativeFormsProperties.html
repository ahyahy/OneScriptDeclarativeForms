ДекларативныеФормы (DeclarativeForms) Свойства

Свойства

cssПуть (cssPath) Возвращает или задает путь до файла стилей css.
nwПуть (NwPath) Возвращает или задает путь до файла nw.exe.
oscriptПуть (OscriptPath) Возвращает или задает путь до файла oscript.exe.
АргументыСобытия (EventArgs) Возвращает аргументы события, связанные с действием, которое вызвало последнее событие.
НоваяСтрока (NewLine) Возвращает строку, обозначающую в данной среде начало новой строки.
ОбщаяСтруктура (ShareStructure) Возвращает объект типа Структура, как глобальное свойство, которое доступно для изменения во всех подключенных сценариях.
Отправитель (Sender) Возвращает источник (объект) последнего события.
Порт (Port) Возвращает или задает порт сервера для взаимодействия со сценариями.
СтрокаФункций (FunctionString) Возвращает или задает строку, содержащую перечисленные через точку с запятой функции js, переданные в ответе сервера.
Форма (Form) Возвращает окно, которое составляет пользовательский интерфейс приложения.
