ДекларативныеФормы.ОбщаяСтруктура (DeclarativeForms.ShareStructure) Свойство

Использование

Только чтение.

Значение

Тип: Структура.

Возвращает объект типа Структура, как глобальное свойство, которое доступно для изменения во всех подключенных сценариях.

Примечание

Пример

ОбщаяСтруктура = ДФ.ОбщаяСтруктура;
