ДекларативныеФормы.cssПуть (DeclarativeForms.cssPath) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает путь до файла стилей css.

Примечание

По умолчанию значение свойства равно строке "styles.css".

Пример

ДФ.cssПуть = "styles.css";
