ТипОтображения (DisplayType) Перечисление

Описание

Указывает тип отображения элемента.

Члены

Имя члена Описание Значение
Блок (Block) Элемент отображается как элемент уровня блока. block
Встроенный (Inline) Элемент отображается как встроенный элемент. Это значение по умолчанию. inline
ВстроенныйБлок (InlineBlock) Элемент отображается как блок внутри встроенного блока. inline-block
ВстроенныйГибкий (InlineFlex) Элемент отображается как гибкий блок встроенного уровня. inline-flex
ВстроенныйСетка (InlineGrid) Элемент отображается как сетка встроенного уровня. inline-grid
ВстроенныйТаблица (InlineTable) Элемент отображается как встроенная таблица (Таблица (Table)) без разрыва строки до или после таблицы. inline-table
Гибкий (Flex) Элемент отображается как гибкий блок на уровне блока. flex
ГруппаКолонокТаблицы (TableColumnGroup) Элемент отображается как группа из одной или нескольких колонок (ГруппаКолонок (Colgroup)). table-column-group
ГруппаСтрокТаблицы (TableRowGroup) Элемент отображается как группа из одной или нескольких строк (ОбластьТаблицы (TableBody)). table-row-group
ЗаголовокГруппыТаблицы (TableHeaderGroup) Элемент отображается как строка заголовка таблицы (ШапкаТаблицы (TableHeader)). table-header-group
ЗаголовокТаблицы (TableCaption) Элемент отображается как заголовок таблицы (Подпись (Label)). table-caption
КолонкаТаблицы (TableColumn) Элемент отображается как колонка ячеек (Колонка (Column)). table-column
Маркер (Marker) Это значение устанавливает содержимое до или после поля в качестве маркера (используется с псевдоэлементами :before и :after. В противном случае это значение идентично Встроенный). marker
НижнийКолонтитулТаблицы (TableFooterGroup) Элемент отображается как строка нижнего колонтитула таблицы (Итоги (Outcome)). table-footer-group
Отсутствие (None) Элемент не будет отображаться. none
Сетка (Grid) Элемент отображается как сетка на уровне блоков. grid
СписокЭлементов (ListItem) Элемент отображается в виде списка. list-item
СтрокаТаблицы (TableRow) Элемент отображается как строка таблицы (СтрокаТаблицы (TableRow)). table-row
Таблица (Table) Элемент отображается как таблица блоков (Таблица (Table)) с разрывом строки до и после таблицы. table
ЯчейкаТаблицы (TableCell) Элемент отображается как ячейка таблицы (Ячейка (Cell) и ЯчейкаЗаголовка (HeaderCell)). table-cell
