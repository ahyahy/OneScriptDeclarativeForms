Элемент.Нажатие (Element.Click) Метод

Синтаксис

Нажатие()

Возвращаемое значение

Описание

Вызывает событие когда элемент щелкнут.

Примечание

Важно: В системе отличной от Windows метод не работает.

Пример

Абзац1.Нажатие();
