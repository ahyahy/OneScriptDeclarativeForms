Элемент.Направление (Element.Dir) Свойство

Использование

Чтение и запись.

Значение

Тип: Направление (Dir).

Возвращает или задает направление расположения элементов внутри элемента.

Примечание

Пример

Контент.Направление = ДФ.Направление.СправаНалево;
