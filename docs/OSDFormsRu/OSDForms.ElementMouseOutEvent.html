Элемент.МышьПокинулаЭлемент (Element.MouseOut) Событие

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код для выполнения, когда указатель мыши покидает пределы элемента.

Примечание

Пример

Блок1.МышьПокинулаЭлемент = ДФ.Действие(ЭтотОбъект, "МышьПокинулаЭлемент");
