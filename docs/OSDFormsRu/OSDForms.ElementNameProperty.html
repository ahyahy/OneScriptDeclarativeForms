Элемент.Имя (Element.Name) Свойство

Использование

Только чтение.

Значение

Тип: Строка.

Возвращает уникальное имя элемента сгенерированное при создании.

Примечание

Пример

Сообщить("Кнопка1.Имя = " + Кнопка1.Имя);
