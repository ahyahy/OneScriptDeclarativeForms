Элемент.ПрокрутитьДо (Element.ScrollIntoView) Метод

Синтаксис

ПрокрутитьДо(<Выравнивание>)

Параметры

Выравнивание (необязательный)
Тип: Булево.
Если Истина - верхняя часть элемента будет выровнена по верхней части видимой области прокручиваемого родителя. Если Ложь - нижняя часть элемента будет выровнена по нижней части видимой области прокручиваемого родителя. Если этот параметр опущен, элемент будет прокручиваться до верхней части элемента.

Возвращаемое значение

Описание

Прокручивает элемент в видимую область окна браузера.

Примечание

Пример

Абзац2.ПрокрутитьДо();
