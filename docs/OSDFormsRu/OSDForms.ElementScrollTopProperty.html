Элемент.ВертикальноеПрокручивание (Element.ScrollTop) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает количество пикселей, на которые содержимое элемента прокручивается по вертикали.

Примечание

Пример

Блок1.ВертикальноеПрокручивание = Блок1.ВертикальноеПрокручивание + 5;
Полный пример кода
