АргументыСобытия.Помечен (EventArgs.Checked) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает состояние флажка для события ПриИзменении (Change).

Примечание

Пример

Процедура Флажок1_ПриИзменении() Экспорт
	Сообщить("ДФ.АргументыСобытия.Помечен = " + ДФ.АргументыСобытия.Помечен);
	Сообщить("Флажок1.Помечен = " + Флажок1.Помечен);
