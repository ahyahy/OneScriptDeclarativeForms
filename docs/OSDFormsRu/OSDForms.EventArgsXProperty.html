АргументыСобытия.Икс (EventArgs.X) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает x-координату щелчка мыши.

Примечание

Пример

ГлавноеМеню.Показать(ДФ.АргументыСобытия.Икс, ДФ.АргументыСобытия.Игрек);
