ПереносГибких (FlexWrap) Перечисление

Описание

Указывает значение, должны ли переноситься гибкие элементы.

Члены

Имя члена Описание Значение
БезПереноса (Nowrap) Гибкие элементы не будут переноситься. Это значение по умолчанию. nowrap
ОбратныйПеренос (WrapReverse) Гибкие элементы при необходимости будут переноситься в обратном порядке. wrap-reverse
Перенос (Wrap) Гибкие элементы будут перенесены при необходимости. wrap
