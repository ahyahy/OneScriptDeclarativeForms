ВариантШрифта (FontVariant) Перечисление

Описание

Указывает значение, должен ли шрифт отображаться маленькими заглавными буквами.

Члены

Имя члена Описание Значение
МаленькиеЗаглавные (SmallCaps) Шрифт отображается маленькими заглавными буквами. small-caps
Нормальный (Normal) Шрифт нормальный. Это значение по умолчанию. normal
