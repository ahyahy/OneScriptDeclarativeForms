Форма.ДобавитьДочерний (Form.AppendChild) Метод

Синтаксис

ДобавитьДочерний(<Элемент>)

Параметры

Элемент (обязательный)
Тип: Произвольный.
Добавляемый в коллекцию элемент.

Возвращаемое значение

Тип: Произвольный.

Добавленный в коллекцию элемент.

Описание

Добавляет элемент в качестве последнего дочернего элемента элемента.

Примечание

Пример

ДополнительныйБлок1 = Форма1.ДобавитьДочерний(ДФ.ДополнительныйБлок());
