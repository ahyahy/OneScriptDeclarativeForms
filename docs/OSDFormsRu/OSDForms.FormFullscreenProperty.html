Форма.Полноэкранный (Form.Fullscreen) Свойство

Использование

Только запись.

Значение

Тип: Булево.

Включить полноэкранный режим.

Будьте осторожны: если для фрейма также установлено значение Ложь в полноэкранном режиме, это предотвратит захват мыши на самых краях экрана. Вам следует избегать его активации, если для полноэкранного режима также установлено значение Истина.

Примечание

Свойство должно быть установлено до открытия формы.

Пример

Форма1.Полноэкранный = Истина;
