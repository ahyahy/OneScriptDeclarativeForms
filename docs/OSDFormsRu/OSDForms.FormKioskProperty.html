Форма.Киоск (Form.Kiosk) Свойство

Использование

Только запись.

Значение

Тип: Булево.

Включить режим киоска.

В режиме киоска приложение будет работать в полноэкранном режиме и будет пытаться предотвратить выход пользователей из приложения, поэтому не забудьте предоставить в приложении возможность выхода из режима киоска. Этот режим в основном используется для презентации на публичных дисплеях.

Примечание

Свойство должно быть установлено до открытия формы.

Из режима Киоск выходят либо alt+tab, либо ctrl+alt+del в windows.

Важно: Не применимо при открытии формы в браузере.

Пример

Форма1.Киоск = Истина;
