Форма (Form) Свойства

Свойства

Заголовок (Title) Заголовок окна при запуске приложения.
Значок (Icon) Путь до иконки.
ИзменяемыйРазмер (Resizable) Логическое значение, которое показывает можно ли пользователю изменять размеры окна.
Киоск (Kiosk) Включить режим киоска.
КлючЭлемента (ItemKey) Возвращает уникальный ключ элемента сгенерированный при создании.
Контент (Body) Возвращает объект Контент (Body), который содержит весь контент, который появится на странице: текст, изображения, теги, скрипты JavaScript и так далее.
МаксимальнаяВысота (MaxHeight) Максимальная внутренняя высота окна.
МаксимальнаяШирина (MaxWidth) Максимальная внутренняя ширина окна.
Меню (Menu) Возвращает или задает строку меню окна.
МинимальнаяВысота (MinHeight) Минимальная внутренняя высота окна.
МинимальнаяШирина (MinWidth) Минимальная внутренняя ширина окна.
НачальнаяВысота (StartHeight) Задает начальную высоту главного окна.
НачальнаяШирина (StartWidth) Задает начальную ширину главного окна.
Позиция (Position) Можно указать позицию окна при загрузке (Неопределено, Центр или Мышь).
Показать (Show) Укажите значение Ложь, если вы хотите, чтобы ваше приложение было скрыто при запуске.
ПоказатьВПанели (ShowInTaskbar) Отображается ли окно на панели задач.
Полноэкранный (Fullscreen) Включить полноэкранный режим.
Прозрачный (Transparent) Сделать окно прозрачным.
Прокручено (ScrolledValue) Возвращает в процентах величину, на которую прокручен ползунок формы.
Фрейм (Frame) Укажите значение Ложь, чтобы сделать окно безрамочным.
Элементы (Children) Возвращает массив, содержащий дочерние элементы формы.
