Композиция (GlobalCompositeOperation) Перечисление

Описание

Указывает способ рисования нового изображения на существующем изображении.

Новое изображение - это то, которое мы собираемся нарисовать. Существующее - то, которое уже нарисовано.

Члены

Имя члена Описание Значение
Новое (Copy) Отображается новое изображение. Существующее изображение игнорируется. copy
НовоеВ (SourceIn) Новое изображение в существующем изображении. Показана только та часть нового изображения, которая находится ВНУТРИ существующего изображения, а существующее изображение прозрачно. source-in
НовоеВне (SourceOut) Новое изображение вне существующего изображения. Показана только та часть нового изображения, которая находится ВНЕ существующего изображения, а существующее изображение прозрачно. source-out
НовоеНад (SourceOver) Новое изображение над существующим изображением. Это значение по умолчанию. source-over
НовоеПоверх (SourceAtop) Новое изображение поверх существующего изображения. Часть нового изображения, находящаяся за пределами существующего изображения, не отображается. source-atop
Объединить (Xor) Новое изображение объединяется с помощью исключающего ИЛИ с существующим изображением. xor
Совместно (Lighter) Отображается новое изображение и существующее изображение. lighter
СуществующееВ (DestinationIn) Существующее изображение отображается в новом изображении. Показана только та часть существующего изображения, которая находится ВНУТРИ нового изображения, а новое изображение прозрачно. destination-in
СуществующееВне (DestinationOut) Существующее изображение отображается вне нового изображения. Показана только та часть существующего изображения, которая находится ВНЕ нового изображения, а новое изображение прозрачно. destination-out
СуществующееНад (DestinationOver) Существующее изображение поверх нового изображения. destination-over
СуществующееПоверх (DestinationAtop) Существующее изображение поверх нового изображения. Часть существующего изображения, находящаяся за пределами нового изображения, не отображается. destination-atop
