ЯчейкаЗаголовка.ДиапазонСтрок (HeaderCell.RowSpan) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает значение, указывающее количество строк, которые должна занимать ячейка таблицы.

Примечание

Пример

ЯчейкаЗаголовка5.ДиапазонСтрок = 2;
Полный пример кода 

Копировать     Выделить всё
