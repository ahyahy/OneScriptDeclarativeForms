ДанныеРисунка.Высота (ImageData.Height) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает высоту прямоугольника на холсте содержащего обрабатываемые пиксели.

Примечание

Пример

Сообщить("ДанныеРисунка1.Высота = " + ДанныеРисунка1.Высота);
