Изображение.ИсходнаяВысота (Image.NaturalHeight) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает исходную высоту изображения.

Примечание

Пример

Процедура Свойство() Экспорт
	Сообщить("Изображение1.ИсходнаяШирина = " + Изображение1.ИсходнаяШирина);
	Сообщить("Изображение1.ИсходнаяВысота = " + Изображение1.ИсходнаяВысота);
