Изображение.ИсходнаяШирина (Image.NaturalWidth) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает исходную ширину изображения.

Примечание

Пример

Процедура ДФ_ПриПолученииСвойства() Экспорт
	Сообщить("Изображение1.ИсходнаяШирина = " + Изображение1.ИсходнаяШирина);
	Сообщить("Изображение1.ИсходнаяВысота = " + Изображение1.ИсходнаяВысота);
