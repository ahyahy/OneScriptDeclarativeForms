ФильтрИзображений (ImagesFilter) Конструктор

Синтаксис

ФильтрИзображений(<Размытие>, <Яркость>, <Контраст>, <Тень>, <ОттенкиСерого>, <ПоворотОттенка>, <Инверсия>, <Непрозрачность>, <Насыщенность>, <Ретро>)

Параметры

Размытие (необязательный)
Тип: Число.
Эффект размытия.
Яркость (необязательный)
Тип: Число.
Яркость изображения.
Контраст (необязательный)
Тип: Число.
Контрастность изображения.
Тень (необязательный)
Тип: Тень (BoxShadow).
Тень изображения.
ОттенкиСерого (необязательный)
Тип: Число.
Оттенки серого.
ПоворотОттенка (необязательный)
Тип: Число.
Поворот оттенка к изображению.
Инверсия (необязательный)
Тип: Число.
Инвертирсия изображения.
Непрозрачность (необязательный)
Тип: Число.
Уровень непрозрачности изображения.
Насыщенность (необязательный)
Тип: Число.
Насыщенность изображения.
Ретро (необязательный)
Тип: Число.
Эффект старинной фотографии.

Описание

Создаёт новый экземпляр класса ФильтрИзображений (ImagesFilter).

Примечание

Пример

ФильтрИзображений1 = ДФ.ФильтрИзображений();
