РасположениеСодержимого (JustifyContent) Перечисление

Описание

Указывает выравнивание между элементами внутри гибкого контейнера, когда элементы не используют все доступное пространство.

Члены

Имя члена Описание Значение
Вконце (FlexEnd) Элементы располагаются в конце контейнера. flex-end
ВНачале (FlexStart) Элементы располагаются в начале контейнера. Это значение по умолчанию. flex-start
Вокруг (SpaceAround) Элементы располагаются с пробелом до, между и после строк. space-around
Вцентре (Center) Элементы располагаются в центре контейнера. center
Между (SpaceBetween) Элементы располагаются с пробелом между строками. space-between
