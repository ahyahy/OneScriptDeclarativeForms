СтильСписка (ListStyle) Конструктор

Синтаксис

СтильСписка(<ТипСтиляСписка>, <ПозицияСтиляСписка>, <КартинкаСтиляСписка>)

Параметры

ТипСтиляСписка (необязательный)
Тип: ТипСтиляСписка (ListStyleType).
Возвращает или задает тип маркера элемента списка.
ПозицияСтиляСписка (необязательный)
Тип: ПозицияСтиляСписка (ListStylePosition).
Возвращает или задает позицию маркера элемента списка.
КартинкаСтиляСписка (необязательный)
Тип: Строка.
Возвращает или задает путь до изображения для маркера элемента списка.

Описание

Создаёт новый экземпляр класса СтильСписка (ListStyle).

Примечание

Пример

НеупорядоченныйСписок1.Стиль.СтильСписка = ДФ.СтильСписка(ДФ.ТипСтиляСписка.ЗакрашенныйКруг, ДФ.ПозицияСтиляСписка.Внутри, "icon.png");
