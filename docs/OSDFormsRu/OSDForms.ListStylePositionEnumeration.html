ПозицияСтиляСписка (ListStylePosition) Перечисление

Описание

Указывает позицию маркера элемента списка.

Члены

Имя члена Описание Значение
Внутри (Inside) Смещает отступ маркера элемента списка. inside
Снаружи (Outside) Маркер элемента списка будет отображаться перед любым текстовым содержимым. Это значение по умолчанию. outside
