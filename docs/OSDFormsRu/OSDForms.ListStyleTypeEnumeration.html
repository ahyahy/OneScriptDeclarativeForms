ТипСтиляСписка (ListStyleType) Перечисление

Описание

Указывает тип маркера элемента списка.

Члены

Имя члена Описание Значение
ВерхнеЛатинский (UpperLatin) Маркер верхнелатинский (A, B, C, D, E и т. д.). upper-latin
ВерхнеРимский (UpperRoman) Маркер верхнеримский (I, II, III, IV, V и т.д.). upper-roman
ЗакрашенныйКруг (Disc) Маркер представляет собой закрашенный круг. Это значение по умолчанию для класса НеупорядоченныйСписок (UnorderedList). disc
Квадрат (Square) Маркер представляет собой квадрат. square
Круг (Circle) Маркер представляет собой круг. circle
НижнеЛатинский (LowerLatin) Маркер нижнелатинский (a, b, c, d, e и т.д.). lower-latin
НижнеРимский (LowerRoman) Маркер нижнеримский (i, ii, iii, iv, v и т.д.). lower-roman
Отсутствие (None) Маркер не отображается. none
Число (Decimal) Маркер представляет собой число. Это значение по умолчанию для класса УпорядоченныйСписок (OrderedList). decimal
ЧислоСНулями (DecimalLeadingZero) Маркер представляет собой число с ведущими нулями (01, 02, 03 и т. д.). decimal-leading-zero
