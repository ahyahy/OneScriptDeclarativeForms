Карта.Области (Map.Areas) Свойство

Использование

Только чтение.

Значение

Тип: Массив.

Возвращает коллекцию всех областей на карте.

Примечание

Пример

Процедура Свойство() Экспорт
	Для А = 0 По Карта1.Области.ВГраница() Цикл
		Сообщить("Адрес области " + А + " = " + Карта1.Области.Получить(А).Адрес);
	КонецЦикла;
