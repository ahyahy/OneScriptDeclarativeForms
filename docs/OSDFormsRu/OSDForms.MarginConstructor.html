Отступ (Margin) Конструктор

Синтаксис

Отступ(<ОтступСверху>, <ОтступСправа>, <ОтступСнизу>, <ОтступСлева>, <Авто>)

Параметры

ОтступСверху (необязательный)
Тип: Число.
Возвращает или задает верхний отступ для элемента.
ОтступСправа (необязательный)
Тип: Число.
Возвращает или задает правый отступ для элемента.
ОтступСнизу (необязательный)
Тип: Число.
Возвращает или задает нижний отступ для элемента.
ОтступСлева (необязательный)
Тип: Число.
Возвращает или задает левый отступ для элемента.
Авто (необязательный)
Тип: Булево.
Возвращает или задает возможность браузеру устанавливать поля (все четыре поля будут равны).

Описание

Создаёт новый экземпляр класса Отступ (Margin).

Примечание

Пример

Абзац2.Стиль.Отступ = ДФ.Отступ(20, 10, 20, 30);
