Отступ (Margin) Свойства

Свойства

Авто (Auto) Возвращает или задает возможность браузеру устанавливать поля (все четыре поля будут равны).
ОтступСверху (MarginTop) Возвращает или задает верхний отступ для элемента.
ОтступСлева (MarginLeft) Возвращает или задает левый отступ для элемента.
ОтступСнизу (MarginBottom) Возвращает или задает нижний отступ для элемента.
ОтступСправа (MarginRight) Возвращает или задает правый отступ для элемента.
