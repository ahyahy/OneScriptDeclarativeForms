ТипМаркера (MarkerType) Перечисление

Описание

Указывает тип маркера для упорядоченного списка.

Члены

Имя члена Описание Значение
Заглавные (Uppercase) Список в алфавитном порядке, заглавные буквы (A, B, C, D). A
РимскиеЗаглавные (RomanUppercase) Римские цифры, заглавные (I, II, III, IV). I
РимскиеСтрочные (RomanLowercase) Римские цифры, строчные (i, ii, iii, iv). i
Строчные (Lowercase) Список в алфавитном порядке, строчные буквы (a, b, c, d). a
Число (Number) Десятичные числа (1, 2, 3, 4). Это значение по умолчанию. 1
