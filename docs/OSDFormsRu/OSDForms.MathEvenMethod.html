Математика.Четное (Math.Even) Метод

Синтаксис

Четное(<ЧисловойПараметр>)

Параметры

ЧисловойПараметр (обязательный)
Тип: Число.
Значение, проверяемое на четность.

Возвращаемое значение

Тип: Булево.

Истина, если число четное; в противном случае, Ложь. Если в параметре передано не целое число, возвращается Неопределено.

Описание

Возвращает значение, которое показывает, является ли проверяемое число четным.

Примечание

|Мат.Четное(-83) = " + Мат.Четное(-83) + "

|";
