Математика.Большее (Math.Max) Метод

Синтаксис

Большее(<Первое>, <Второе>)

Параметры

Первое (обязательный)
Тип: Число.
Первое из двух сравниваемых чисел.
Второе (обязательный)
Тип: Число.
Второе из двух сравниваемых чисел.

Возвращаемое значение

Тип: Число.

Большее из значений двух параметров Первое и Второе.

Описание

Возвращает большее из двух сравниваемых чисел.

Примечание

Пример

Форма1.Контент.Текст = "" + ДФ.Математика().Большее(10, 3);
