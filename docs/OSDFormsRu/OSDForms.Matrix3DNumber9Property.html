Матрица3Д.Число9 (Matrix3D.Number9) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает девятое значение для матрицы 3D преобразования.

Примечание

Пример

Матрица3Д2.Число9 = 8;
