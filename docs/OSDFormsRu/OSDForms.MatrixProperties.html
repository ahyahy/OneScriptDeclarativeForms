Матрица (Matrix) Свойства

Свойства

МасштабИгрек (ScaleY) Возвращает или задает масштабирование по горизонтали. Значение меньше 1 уменьшает элемент, больше 1 - увеличивает.
МасштабИкс (ScaleX) Возвращает или задает масштабирование по горизонтали. Значение от 0 до 1 уменьшает элемент, больше 1 - увеличивает.
СдвигИгрек (TransformY) Возвращает или задает деформацию (сдвиг) стороны элемента по оси Y, положительное значение - вверх, отрицательное - вниз.
СдвигИкс (TransformX) Возвращает или задает деформацию (сдвиг) стороны элемента по оси X, положительное значение - влево, отрицательное - вправо.
СмещениеИгрек (OffsetY) Возвращает или задает смещение по вертикали. Положительное значение - вниз, отрицательное - вверх.
СмещениеИкс (OffsetX) Возвращает или задает смещение по горизонтали. Положительное значение - вправо, отрицательное - влево.
