ЭлементМеню.Нажатие (MenuItem.Click) Событие

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код для выполнения при нажатии элемента меню.

Примечание

Пример

Сохранить.Нажатие = ДФ.Действие(ЭтотОбъект, "Сохранить_Нажатие");
