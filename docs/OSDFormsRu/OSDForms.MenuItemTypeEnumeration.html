ТипЭлементаМеню (MenuItemType) Перечисление

Описание

Указывает тип элемента меню при создании.

Члены

Имя члена Описание Значение
Разделитель (Separator) Будет создан элемент меню разделитель. separator
Стандартный (Normal) Будет создан стандартный элемент меню. normal
Флажок (Checkbox) Будет создан элемент меню флажок. checkbox
