Меню (Menu) Методы

Методы

Вставить (Insert) Вставляет элемент меню в указанную позицию меню.
Добавить (Append) Добавляет элемент в конец меню.
Показать (Popup) Показывает контекстное меню по привязке (x, y) в текущем окне.
Удалить (Remove) Удаляет пункт из меню.
УдалитьПоИндексу (RemoveAt) Удаляет пункт из меню.
