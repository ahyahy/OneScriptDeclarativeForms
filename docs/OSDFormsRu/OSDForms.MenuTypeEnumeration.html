ТипМеню (MenuType) Перечисление

Описание

Указывает тип меню при создании.

Члены

Имя члена Описание Значение
Главное (Menubar) Будет создано главное меню. menubar
Контекстное (Contextmenu) Будет создано контекстное меню. contextmenu
Отсутствие (None) Будет создано меню для создания набора подэлементов. none
