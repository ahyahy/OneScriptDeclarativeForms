ОкноСообщений (MessageBox) Конструктор

Синтаксис

ОкноСообщений()

ОкноСообщений(<Заголовок>, <Текст>, <Позиция>)

Параметры

Заголовок (необязательный)
Тип: Строка.
Текст заголовка.
Текст (необязательный)
Тип: Строка.
Текст окна сообщений.
Позиция (необязательный)
Тип: Точка (Point).
Положение окна сообщений на экране при открытии. Точка нахождения левого верхнего угла окна.

Описание

Создаёт новый экземпляр класса ОкноСообщений (MessageBox).

Примечание

Важно: При установке свойства ДекларативныеФормы.ОткрытьВБраузере (DeclarativeForms.OpenInBrowser) в значение Истина этот элемент управления не работает.

Важно: В ОС Linux включите группировку кнопок в панели задач, чтобы избежать появления там множества кнопок от каждого нового сообщения.

Пример

ОкноСообщений1 = ДФ.ОкноСообщений();
