ВыборМесяца (MonthSelection) Методы

Методы

Анимация (Animation) Позволяет определять и управлять анимациями элементов.
Выбрать (Select) Выбирает содержимое текстового поля месяца.
ДвойноеНажатие (DoubleClick) Вызывает событие когда элемент дважды щелкнут.
Нажатие (Click) Вызывает событие когда элемент щелкнут.
ПрокрутитьДо (ScrollIntoView) Прокручивает элемент в видимую область окна браузера.
СнятьФокус (Blur) Удаляет фокус с элемента.
Удалить (Remove) Удаляет элемент.
Фокус (Focus) Устанавливает фокус на элементе.
ШагВверх (StepUp) Увеличивает значение поля месяца на указанное число.
ШагВниз (StepDown) Уменьшает значение поля месяца на указанное число.
