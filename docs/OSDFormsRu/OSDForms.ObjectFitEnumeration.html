ВписываниеОбъекта (ObjectFit) Перечисление

Описание

Указывает как содержимое элемента должно помещаться в поле, заданное по высоте и ширине.

Члены

Имя члена Описание Значение
Вместить (Contain) Содержимое масштабируется для сохранения соотношения сторон и при этом помещается в заданное поле. contain
Заполнение (Fill) Размер содержимого соответствует размеру заданного поля. При необходимости объект будет растянут или сжат по размеру. Это значение по умолчанию. fill
Масштабировать (ScaleDown) Содержимое имеет размер, приведенный к меньшему размеру заданной области. scale-down
Накрыть (Cover) Содержимого изменяется таким образом, чтобы сохранить соотношение сторон при заполнении всего заданного поля. Объект будет обрезан по размеру. cover
Отсутствие (None) Размер содержимого не изменяется. none
