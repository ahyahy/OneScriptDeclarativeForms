УпорядоченныйСписок (OrderedList) Класс

Описание

Представляет элемент HTML <ol>.

Создаёт нумерованный список элементов. Используется для вёрстки перечня однотипных элементов, когда важен их порядок. Можно задать число, с которого будет начинаться нумерованный список.
