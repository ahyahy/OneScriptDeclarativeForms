Переполнение (Overflow) Перечисление

Описание

Указывает поведение элемента при выходе содержимого за поля элемента.

Члены

Имя члена Описание Значение
Авто (Auto) Содержимое обрезается, и при необходимости добавляются полосы прокрутки. auto
Отображено (Visible) Содержимое НЕ обрезается и может отображаться за пределами поля элемента. Это значение по умолчанию. visible
Прокрутка (Scroll) Добавляются полосы прокрутки, а содержимое обрезается при необходимости. scroll
Скрыто (Hidden) Содержимое за пределами поля элемента не отображается. hidden
