ПовторШаблона (PatternRepeat) Перечисление

Описание

Указывает варианты повтора шаблона.

Члены

Имя члена Описание Значение
НеПовторять (NoRepeat) Рисунок отобразится только один раз, без повтора. no-repeat
Повторять (Repeat) Рисунок повторяется как по горизонтали, так и по вертикали. Это значение по умолчанию. repeat
ПовторятьВертикально (RepeatY) Рисунок повторяется только вертикально. repeat-y
ПовторятьГоризонтально (RepeatX) Рисунок повторяется только по горизонтали. repeat-x
