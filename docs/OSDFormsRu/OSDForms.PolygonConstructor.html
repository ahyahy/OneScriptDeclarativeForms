Полигон (Polygon) Конструктор

Синтаксис

Полигон(<Координаты>)

Параметры

Координаты (необязательный)
Тип: Массив.
Массив координат краев многоугольника состоящий из объектов Точка (Point).

Описание

Создаёт новый экземпляр класса Полигон (Polygon).

Примечание

Пример

Полигон1 = ДФ.Полигон();
