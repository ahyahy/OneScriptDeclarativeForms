Позиция (Position) Перечисление

Описание

Указывает тип позиционирования для элемента.

Члены

Имя члена Описание Значение
Абсолютно (Absolute) Позиционирование относительно своего первого позиционированного (не статического) элемента-предка. absolute
Относительно (Relative) Позиционирование относительно своего обычного положения. relative
Статично (Static) Элементы отображаются в порядке появления в потоке документов. Это значение по умолчанию. static
Стикер (Sticky) Позиционирование в зависимости от позиции полосы прокрутки. Прикрепленный элемент переключается между относительным и фиксированным положением в зависимости от положения полосы прокрутки. Он позиционируется относительно до тех пор, пока в окне просмотра не будет достигнута заданная позиция смещения - тогда он приклеивается на месте. sticky
Фиксировано (Fixed) Позиционирование относительно окна браузера. fixed
