Индикатор.Позиция (Progress.Position) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает текущую позицию индикатора.

Примечание

Результат деления текущего значения на максимальное значение.

Пример

Сообщить("Индикатор1.Позиция = " + Индикатор1.Позиция);
Полный пример кода
