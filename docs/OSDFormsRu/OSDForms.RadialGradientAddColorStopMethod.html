РадиальныйГрадиент.ДобавитьОстановкуГрадиента (RadialGradient.AddColorStop) Метод

Синтаксис

ДобавитьОстановкуГрадиента(<Стоп>, <Цвет>)

Параметры

Стоп (обязательный)
Тип: Число.
Значение от 0 до 1. Положение между началом и концом градиента.
Цвет (обязательный)
Тип: Цвет (Color).
Цвет для отображения в позиции остановки.

Возвращаемое значение

Описание

Определяет цвет и положение в объекте градиента.

Примечание

Пример

РадиальныйГрадиент1.ДобавитьОстановкуГрадиента(0, ДФ.Цвет.Красный);
