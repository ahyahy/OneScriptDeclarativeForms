Переключатель()

Параметры

Описание

Создаёт новый экземпляр класса Переключатель (Radio).

Примечание

Чтобы объединить несколько переключателей в группу, задайте им одинаковое значение свойства Имя (Name).

Пример

Переключатель1 = ДополнительныйБлок1.ДобавитьДочерний(ДФ.Переключатель());
