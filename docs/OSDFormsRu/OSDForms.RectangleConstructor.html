Прямоугольник (Rectangle) Конструктор

Синтаксис

Прямоугольник(<X>, <Y>, <Ширина>, <Высота>)

Параметры

X (необязательный)
Тип: Число.
Y (необязательный)
Тип: Число.
Ширина (необязательный)
Тип: Число.
Высота (необязательный)
Тип: Число.

Описание

Создаёт новый экземпляр класса Прямоугольник (Rectangle).

Примечание

Пример

Прямоугольник1 = ДФ.Прямоугольник();
Прямоугольник2 = ДФ.Прямоугольник(10, 10, 100, 150);
