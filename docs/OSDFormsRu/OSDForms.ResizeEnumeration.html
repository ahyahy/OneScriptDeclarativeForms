ИзменяемыйРазмер (Resize) Перечисление

Описание

Указывает значение, может ли пользователь изменить размер элемента.

Члены

Имя члена Описание Значение
Высота (Vertical) Пользователь может регулировать высоту элемента. vertical
Отсутствие (None) Пользователь не может изменить размер элемента. Это значение по умолчанию. none
Ширина (Horizontal) Пользователь может регулировать ширину элемента. horizontal
ШиринаВысота (Both) Пользователь может регулировать как высоту, так и ширину элемента. both
