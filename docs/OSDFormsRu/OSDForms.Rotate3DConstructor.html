Поворот3Д (Rotate3D) Конструктор

Синтаксис

Поворот3Д(<Икс>, <Игрек>, <Зет>, <Угол>)

Параметры

Икс (необязательный)
Тип: Число.
Значение по оси X для преобразования 3D-поворот (от 0 до 1).
Игрек (необязательный)
Тип: Число.
Значение по оси Y для преобразования 3D-поворот (от 0 до 1).
Зет (необязательный)
Тип: Число.
Значение по оси Z для преобразования 3D-поворот (от 0 до 1).
Угол (необязательный)
Тип: Число.
Величина угла для преобразования 3D-поворот.

Описание

Создаёт новый экземпляр класса Поворот3Д (Rotate3D).

Примечание

Пример

Поворот3Д1 = ДФ.Поворот3Д();
Поворот3Д2 = ДФ.Поворот3Д(16, 18, 32, 30);
