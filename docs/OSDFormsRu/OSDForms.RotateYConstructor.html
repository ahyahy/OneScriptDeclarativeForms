ПоворотИгрек (RotateY) Конструктор

Синтаксис

ПоворотИгрек(<Угол>)

Параметры

Угол (необязательный)
Тип: Число.
Угол 3D вращения вдоль оси Y.

Описание

Создаёт новый экземпляр класса ПоворотИгрек (RotateY).

Примечание

Пример

ПоворотИгрек1 = ДФ.ПоворотИгрек();
ПоворотИгрек2 = ДФ.ПоворотИгрек(16);
