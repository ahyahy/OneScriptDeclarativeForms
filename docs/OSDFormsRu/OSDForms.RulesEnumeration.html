Разлиновка (Rules) Перечисление

Описание

Указывает как отображать границы между ячейками.

Члены

Имя члена Описание Значение
Все (All) Граница рисуется вокруг каждой ячейки таблицы. all
Группы (Groups) Граница рисуется между группами, которые образуются элементами ШапкаТаблицы (TableHeader), Итоги (Outcome), ОбластьТаблицы (TableBody), ГруппаКолонок (Colgroup) или Колонка (Column). groups
Колонки (Cols) Граница рисуется между колонками. cols
Отсутствие (None) Все границы скрываются. none
Строки (Rows) Граница рисуется между строками таблицы, созданными через элемент СтрокаТаблицы (TableRow). rows
