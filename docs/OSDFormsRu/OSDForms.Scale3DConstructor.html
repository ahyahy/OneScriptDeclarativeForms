Масштаб3Д (Scale3D) Конструктор

Синтаксис

Масштаб3Д(<Икс>, <Игрек>, <Зет>)

Параметры

Икс (необязательный)
Тип: Число.
Значение по оси X для преобразования 3D-масштаб.
Игрек (необязательный)
Тип: Число.
Значение по оси Y для преобразования 3D-масштаб.
Зет (необязательный)
Тип: Число.
Значение по оси Z для преобразования 3D-масштаб.

Описание

Создаёт новый экземпляр класса Масштаб3Д (Scale3D).

Примечание

Пример

Масштаб3Д1 = ДФ.Масштаб3Д();
Масштаб3Д2 = ДФ.Масштаб3Д(2.3, 4.2, 3.1);
