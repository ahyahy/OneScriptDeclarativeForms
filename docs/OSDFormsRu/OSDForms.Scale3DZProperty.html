Масштаб3Д.Зет (Scale3D.Z) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Значение по оси Z для преобразования 3D-масштаб.

Примечание

Пример

Масштаб3Д1.Зет = 3.1;
