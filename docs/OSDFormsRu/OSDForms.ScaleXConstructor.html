МасштабИкс (ScaleX) Конструктор

Синтаксис

МасштабИкс(<Икс>)

Параметры

Икс (необязательный)
Тип: Число.
Значение для оси X.

Описание

Создаёт новый экземпляр класса МасштабИкс (ScaleX).
