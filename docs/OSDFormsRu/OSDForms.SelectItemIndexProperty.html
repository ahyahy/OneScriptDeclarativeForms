ЭлементПоляСписка.Индекс (SelectItem.Index) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает индекс элемента.

Примечание

Пример

Процедура Свойство() Экспорт
	Сообщить("ЭлементПоляСписка3.Индекс = " + ЭлементПоляСписка3.Индекс);
