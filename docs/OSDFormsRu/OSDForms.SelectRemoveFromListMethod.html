ПолеСписка.УдалитьИзСписка (Select.RemoveFromList) Метод

Синтаксис

УдалитьИзСписка(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Указывает индекс удаляемого элемента. Индекс начинается с 0.

Возвращаемое значение

Описание

Удаляет элемент из поля списка.

Примечание

Пример

ПолеСписка1.УдалитьИзСписка(2);
