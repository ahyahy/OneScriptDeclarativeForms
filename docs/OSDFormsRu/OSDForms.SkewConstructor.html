Наклон (Skew) Конструктор

Синтаксис

Наклон(<УголИкс>, <УголИгрек>)

Параметры

УголИкс (необязательный)
Тип: Число.
Величина угла при 2D преобразовании наклона вдоль оси X.
УголИгрек (необязательный)
Тип: Число.
Величина угла при 2D преобразовании наклона вдоль оси Y.

Описание

Создаёт новый экземпляр класса Наклон (Skew).

Примечание

Пример

Наклон1 = ДФ.Наклон();
