Стиль.СмещениеКартинкиГраницы (Style.BorderImageOutset) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает величину, на которую область рисунка границы выходит за границы.

Примечание

Пример

Блок1.Стиль.СмещениеКартинкиГраницы = 0;
Полный пример кода 

Копировать     Выделить всё

ДФ = Новый ДекларативныеФормы();
Форма1 = ДФ.Форма;
