Стиль.КоличествоКолонок (Style.ColumnCount) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает количество колонок, на которые должен быть разделен элемент.

Примечание

Пример

Абзац1.Стиль.КоличествоКолонок = 3;
