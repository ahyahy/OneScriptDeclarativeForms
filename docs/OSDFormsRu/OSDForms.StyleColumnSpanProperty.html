Стиль.ДиапазонКолонокЭлемента (Style.ColumnSpan) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает количество колонок, которые должен охватывать элемент. Значение по умолчанию равно 1.

Примечание

Вариантов выбора только два. Это 1 и -1.

При значении -1 будут охвачены все колонки.

Пример

Заголовок2.Стиль.ДиапазонКолонокЭлемента = -1;
