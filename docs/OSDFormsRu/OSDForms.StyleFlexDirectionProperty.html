Стиль.НаправлениеЭлементов (Style.FlexDirection) Свойство

Использование

Чтение и запись.

Значение

Тип: НаправлениеЭлементов (FlexDirection).

Возвращает или задает направление элементов внутри флекс-контейнера.

Примечание

Пример

Блок1.Стиль.НаправлениеЭлементов = ДФ.НаправлениеЭлементов.КолонкаОбратно;
