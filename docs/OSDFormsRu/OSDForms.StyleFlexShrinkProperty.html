Стиль.Уменьшение (Style.FlexShrink) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает, насколько элемент уменьшится относительно остальных.

Примечание

Пример

Стиль1.Уменьшение = 0;
