Стиль.ОтступСправа (Style.MarginRight) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает правый отступ для элемента.

Примечание

Пример

Кнопка1.Стиль.ОтступСправа = 50;
