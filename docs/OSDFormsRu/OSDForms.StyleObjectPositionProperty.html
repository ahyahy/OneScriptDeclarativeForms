Стиль.ПозицияОбъекта (Style.ObjectPosition) Свойство

Использование

Чтение и запись.

Значение

Тип: Точка (Point).

Возвращает или задает положение элемента внутри поля содержимого. Первое значение - ось X, второе значение - ось Y.

Примечание

Пример

Изображение1.Стиль.ПозицияОбъекта = ДФ.Точка(20, 10);
