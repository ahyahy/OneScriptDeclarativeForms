Стиль.ИсточникПерспективы (Style.PerspectiveOrigin) Свойство

Использование

Чтение и запись.

Значение

Тип: ИсточникПерспективы (PerspectiveOrigin).

Возвращает или задает где находится трехмерный элемент по осям x и y (от 0 до 1).

Примечание

Пример

ИсточникПерспективы1 = ДФ.ИсточникПерспективы(0.5, 0.9);
Блок1.Стиль.ИсточникПерспективы = ИсточникПерспективы1;
