РазмещениеВТаблице (TableLayout) Перечисление

Описание

Указывает способ расположения ячеек, строк и колонок таблицы.

Члены

Имя члена Описание Значение
Авто (Auto) Ширина колонки задается максимально широким неразрывным содержимым. Это значение по умолчанию. auto
Фиксировано (Fixed) Ширина колонки определяется шириной таблицы и колонок, а не содержимым ячеек. fixed
