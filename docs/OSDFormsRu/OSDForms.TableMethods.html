Таблица (Table) Методы

Методы

ВставитьСтроку (InsertRow) Создает пустую строку и добавляет её в таблицу.
ДобавитьДочерний (AppendChild) Добавляет элемент в качестве последнего дочернего элемента элемента.
ПрокрутитьДо (ScrollIntoView) Прокручивает элемент в видимую область окна браузера.
СнятьФокус (Blur) Удаляет фокус с элемента.
Удалить (Remove) Удаляет элемент.
УдалитьДочерний (RemoveChild) Удаляет дочерний элемент элемента.
УдалитьЗаголовок (DeleteCaption) Удаляет первый объект Заголовок (Caption) из таблицы.
УдалитьИтоги (DeleteTFoot) Удаляет объект Итоги (Outcome) из таблицы.
УдалитьСтроку (DeleteRow) Удаляет строку из таблицы.
УдалитьШапку (DeleteTHead) Удаляет объект ШапкаТаблицы (TableHeader) из таблицы.
Фокус (Focus) Устанавливает фокус на элементе.
