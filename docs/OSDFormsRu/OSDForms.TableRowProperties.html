СтрокаТаблицы (TableRow) Свойства

Свойства

ВертикальноеПрокручивание (ScrollTop) Возвращает или задает количество пикселей, на которые содержимое элемента прокручивается по вертикали.
ГоризонтальноеПрокручивание (ScrollLeft) Возвращает или задает количество пикселей, на которые содержимое элемента прокручивается по горизонтали.
Идентификатор (Id) Возвращает или задает идентификатор элемента.
Имя (Name) Возвращает уникальное имя элемента сгенерированное при создании.
ИмяКласса (ClassName) Возвращает или задает имя класса элемента.
ИндексСтроки (RowIndex) Возвращает позицию строки в коллекции строк таблицы.
ИндексСтрокиВСекции (SectionRowIndex) Возвращает позицию строки в коллекции строк объектов ОбластьТаблицы (TableBody), ШапкаТаблицы (TableHeader) или Итоги (Outcome).
КлавишаДоступа (AccessKey) (Элемент) Возвращает или задает горячую клавишу для активации или фокусировки элемента.
Направление (Dir) Возвращает или задает направление расположения элементов внутри элемента.
ПорядокОбхода (TabIndex) Возвращает или задает последовательность перехода по клавише TAB между элементами.
Редактируемый (ContentEditable) Возвращает или задает, доступно содержимое элемента для редактирования или нет.
Родитель (Parent) Возвращает или задает родителя для элемента управления.
Текст (Text) Возвращает или задает текстовое содержимое элемента.
Элементы (Children) Возвращает массив, содержащий дочерние элементы этого элемента.
Ячейки (Cells) Возвращает коллекцию всех ячеек в строке таблицы.
