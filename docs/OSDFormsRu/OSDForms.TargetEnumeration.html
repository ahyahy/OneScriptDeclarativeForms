Назначение (Target) Перечисление

Описание

Указывает где открыть связанный по ссылке документ.

Члены

Имя члена Описание Значение
Новое (Blank) Открывает документ в новом окне. _blank
Развернуто (Top) Открывает документ в развернутом окне. _top
Родитель (Parent) Открывает документ в родительском наборе фреймов. _parent
Фрейм (FrameName) Открывает документ в именованном фрейме. framename
ЭтоОкно (Self) Открывает документ в том же окне. Это значение по умолчанию. _self
