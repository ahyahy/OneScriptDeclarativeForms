ВыравниваниеТекста (TextAlign) Перечисление

Описание

Указывает выравнивание текстового содержимого.

Члены

Имя члена Описание Значение
Конец (End) Выравнивается по конечной позиции. end
Лево (Left) Выравнивается слева. left
Начало (Start) Выравнивается по начальной позиции. start
Право (Right) Выравнивается справа. right
Центр (Center) Выравнивается по центру. center
