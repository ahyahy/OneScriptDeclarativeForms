ОформлениеТекстаСтиль (TextDecorationStyle) Перечисление

Описание

Указывает стиль линии в текстовом оформлении.

Члены

Имя члена Описание Значение
Волнистая (Wavy) Линия будет отображаться в виде волнистой линии. wavy
Двойная (Double) Линия будет отображаться как двойная линия. double
Сплошная (Solid) Линия будет отображаться как одна строка. Это значение по умолчанию. solid
ТочкаПробел (Dotted) Линия будет отображаться в виде пунктирной линии. dotted
ЧертаПробел (Dashed) Линия будет отображаться в виде пунктирной линии. dashed
