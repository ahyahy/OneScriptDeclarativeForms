ПереполнениеТекста (TextOverflow) Перечисление

Описание

Указывает то, что должно произойти, когда текст переполняет содержащий его элемент.

Члены

Имя члена Описание Значение
Многоточие (Ellipsis) Отображение многоточия для представления обрезанного текста. ellipsis
Обрезка (Clip) Текст будет обрезан. Это значение по умолчанию. clip
Строка (String) Отобразить строку для представления обрезанного текста. string
