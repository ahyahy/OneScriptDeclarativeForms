ПрописныеТекста (TextTransform) Перечисление

Описание

Указывает прописные (заглавные) буквы текста.

Члены

Имя члена Описание Значение
ВсеПрописные (Uppercase) Все символы преобразуются в верхний регистр. uppercase
ВсеСтрочные (Lowercase) Все символы преобразуются в нижний регистр. lowercase
Отсутствие (None) Символы не трансформируются. Это значение по умолчанию. none
ПерваяПрописная (Capitalize) Первый символ каждого слова преобразуется в верхний регистр. capitalize
