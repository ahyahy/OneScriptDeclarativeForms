Таймер.Активен (Timer.Enabled) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает признак активности таймера.

Примечание

Пример

Блок1.Текст = "" + ТекущаяУниверсальнаяДатаВМиллисекундах() + "
|Таймер активен = " + Таймер1.Активен;
