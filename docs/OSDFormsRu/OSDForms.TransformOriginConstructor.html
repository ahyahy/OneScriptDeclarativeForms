ТочкаСдвига (TransformOrigin) Конструктор

Синтаксис

ТочкаСдвига(<ПоложениеИкс>, <ПоложениеИгрек>, <ПоложениеЗет>)

Параметры

ПоложениеИкс (необязательный)
Тип: Число.
Положение по оси X.
ПоложениеИгрек (необязательный)
Тип: Число.
Положение по оси Y.
ПоложениеЗет (необязательный)
Тип: Число.
Положение по оси Z.

Описание

Создаёт новый экземпляр класса ТочкаСдвига (TransformOrigin).

Примечание

Пример

ТочкаСдвига1 = ДФ.ТочкаСдвига();
ТочкаСдвига2 = ДФ.ТочкаСдвига(15, 25, 30);
