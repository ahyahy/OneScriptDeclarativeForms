СтильСдвига (TransformStyle) Перечисление

Описание

Указывает способ отображения вложенных элементов в трехмерном пространстве.

Члены

Имя члена Описание Значение
Плоский (Flat) Дочерние элементы НЕ сохранят свое трехмерное положение. Это значение по умолчанию. flat
Трехмерный (Preserve3d) Дочерние элементы сохранят свое трехмерное положение. preserve-3d
