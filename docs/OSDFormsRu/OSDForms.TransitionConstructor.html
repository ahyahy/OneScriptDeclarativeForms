Переход (Transition) Конструктор

Синтаксис

Переход(<СвойствоПерехода>, <ВремяПерехода>, <ФункцияПерехода>, <ЗадержкаПерехода>)

Параметры

СвойствоПерехода (необязательный)
Тип: СвойствоПерехода (TransitionProperty).
Возвращает или задает свойство для которого предназначен эффект перехода.
ВремяПерехода (необязательный)
Тип: Число.
Возвращает или задает количество секунд или миллисекунд, необходимое для завершения эффекта перехода.
ФункцияПерехода (необязательный)
Тип: ФункцияПерехода (TransitionTimingFunction); БезьеКуб (CubicBezier).
Возвращает или задает кривую скорости эффекта перехода.
ЗадержкаПерехода (необязательный)
Тип: Число.
Возвращает или задает время начала эффекта перехода.

Описание

Создаёт новый экземпляр класса Переход (Transition).

Примечание

Пример

Переход1 = ДФ.Переход();
Переход2 = ДФ.Переход(ДФ.СвойствоПерехода.Все, 5, ДФ.ФункцияПерехода.Линейно, 4);
