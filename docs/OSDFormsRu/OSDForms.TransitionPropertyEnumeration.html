СвойствоПерехода (TransitionProperty) Перечисление

Описание

Указывает свойство для которого предназначен эффект перехода.

Члены

Имя члена Описание Значение
Все (All) Все свойства получат эффект перехода. Это значение по умолчанию. all
Отсутствие (None) Ни один объект не получит эффект перехода. none
