Перевод (Translate) Конструктор

Синтаксис

Перевод(<Икс>, <Игрек>)

Параметры

Икс (необязательный)
Тип: Число.
Значение по оси X для преобразования 2D-перевод.
Игрек (необязательный)
Тип: Число.
Значение по оси Y для преобразования 2D-перевод.

Описание

Создаёт новый экземпляр класса Перевод (Translate).

Примечание

Пример

Перевод1 = ДФ.Перевод();
