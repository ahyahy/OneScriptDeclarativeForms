Лоток.Меню (Tray.Menu) Свойство

Использование

Чтение и запись.

Значение

Тип: Меню (Menu).

Возвращает или задает меню для элемента управления Лоток (Tray).

Примечание

Важно: Не применимо при открытии формы в браузере.

Пример

Лоток1.Меню = ГлавноеМеню;
