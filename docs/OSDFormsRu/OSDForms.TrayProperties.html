Лоток (Tray) Свойства

Свойства

Заголовок (Title) Возвращает или задает заголовок для элемента управления Лоток (Tray).
Значок (Icon) Возвращает или задает путь к файлу значка.
Меню (Menu) Возвращает или задает меню для элемента управления Лоток (Tray).
Подсказка (Tooltip) Возвращает или задает всплывающую подсказку лотка.
