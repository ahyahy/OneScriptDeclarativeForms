Лоток.Заголовок (Tray.Title) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает заголовок для элемента управления Лоток (Tray).

Примечание

Важно: Не применимо при открытии формы в браузере.

Пример

Лоток1.Заголовок = "Заголовок";
