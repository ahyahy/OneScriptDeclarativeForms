ВыделениеПользователем (UserSelect) Перечисление

Описание

Указывает значение, можно ли выделить текст элемента.

Члены

Имя члена Описание Значение
Авто (Auto) Текст можно выбрать в соответствии с настройками браузера по умолчанию. Это значение по умолчанию. auto
Все (All) Выбор текста осуществляется одним щелчком мыши вместо двойного щелчка. all
Отсутствие (None) Запретить выделение текста. none
Текст (Text) Текст может быть выбран пользователем. text
