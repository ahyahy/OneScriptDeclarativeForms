Видимость (Visibility) Перечисление

Описание

Указывает значение, должен ли элемент быть видимым.

Члены

Имя члена Описание Значение
Видимый (Visible) Элемент виден. Это значение по умолчанию. visible
Свернуто (Collapse) При использовании в строке или ячейке таблицы элемент не виден. collapse
Скрыто (Hidden) Элемент не виден, но все равно влияет на макет. hidden
