Пробелы (WhiteSpace) Перечисление

Описание

Возвращает или задает способ обработки табуляции, разрывов строк и пробелов в тексте.

Члены

Имя члена Описание Значение
ЗаменятьПереносить (PreLine) Последовательности пробелов заменяются на один пробел. Текст будет переноситься при необходимости. pre-line
НеПереносить (Nowrap) Последовательности пробелов заменяются на один пробел. Текст никогда не переносится на следующую строку. nowrap
Сохранять (Pre) Пробелы сохраняются. pre
СохранятьПереносить (PreWrap) Пробелы сохраняются. Текст будет переноситься при необходимости. pre-wrap
Стандартно (Normal) Последовательности пробелов заменяются на один пробел. Текст будет перенесен при необходимости. Это значение по умолчанию. normal
